Advent of Code 2020
===================

![Check](https://github.com/MortalFlesh/advent-of-code-2020/workflows/Check/badge.svg)

> A console app solving an [Advent of Code 2020](https://adventofcode.com/2020)

---
### Development

First run:
```
paket install
./build.sh
```

or `./build.sh -t Watch`

List commands
```sh
bin/console list
```

Run tests locally
```sh
./build.sh -t Tests
```
